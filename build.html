What is this?

This is a buildserv. if you want to keep track of the development of The Powder Toy you can use the binaries here.

Please keep in mind you can't publish saves using these, currently only 32bit builds are available.

All binaries present on this site are built nightly by an automated system. The binaries might not even run at all. They might trip your grandmother. Use at your own risk.

Just click the status image to start downloading. You can still download the last working build if the current build failed by clicking the status image.
